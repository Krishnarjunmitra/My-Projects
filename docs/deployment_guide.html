Deployment Guide

Step 1: Prepare Environment

Before deploying the application, ensure that you have the necessary environment set up:

  • Ensure Docker is installed on your deployment server.
  • Make sure you have Docker Compose installed.
  • Ensure the server has sufficient resources to run the application.
Step 2: Clone Repository

Clone the project repository from GitHub:

git clone https://github.com/Krishnarjunmitra/CNMMS.git

Navigate to the project directory:

cd CNMMS
Step 3: Build and Run Containers

Build and run Docker containers using Docker Compose:

docker-compose up --build

This command will build the necessary Docker images and start the containers.

Step 4: Access the Application

Once the containers are running, you can access the application using a web browser:

http://your-server-ip:3000

Replace "your-server-ip" with the actual IP address of your deployment server.

Step 5: Deploy Frontend on Netlify

To deploy the frontend on Netlify, follow these steps:

  • Login to Netlify and create a new site.
  • Connect to your GitHub repository and select the appropriate branch.
  • Configure the build settings:
    • Setting Value
    Branch to deploy master
    Build command npm run build
    Publish directory frontend/build
  • Click "Deploy site" to start the deployment process.

Once deployed, you will receive a link to access your live site.
