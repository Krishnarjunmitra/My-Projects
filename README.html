Corporate Network Monitoring and Management System (CNMMS)

Project Overview

Objective: Development of a comprehensive network monitoring and management system for a corporate environment.

Key Features:

  • Network Traffic Monitoring: It can capture and analyze network traffic in real-time.
  • Anomaly Detection: It is able to identify unusual network activities that may indicate security threats.
  • Network Performance Metrics: It can track and report on key performance indicators (KPIs) like bandwidth usage, latency, and packet loss.
  • Dashboard: A simple web-based interface for network administrators to visualize network status and perform management tasks.
  • Alert System: Notify administrators of critical network events.

Installation Guide

Prerequisites:

  • Python 3.8+
  • Node.js 14+
  • Docker

Backend Installation

  1. Clone the repository:
    2. git clone https://github.com/yourusername/CNMMS.git
    cd CNMMS/backend
  2. Create a virtual environment and activate it:
    3. python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
  3. Install dependencies:
    4. pip install -r requirements.txt

Frontend Installation

  1. Navigate to the frontend directory:
    2. cd ../frontend
  2. Install dependencies:
    3. npm install

Deployment Process

Using Docker:

  1. Navigate to the project root:
    2. cd CNMMS
  2. Build and run the Docker containers:
    3. docker-compose up --build
